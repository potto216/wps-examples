# Use Cases

This directory contains higher-level workflows and demos built on top of the capture, analysis, and export tooling in this repository.

## Available examples

- `usecase/llm_rag_ml/README.md`: LLM-guided statistical analyst demo for BLE + Wi-Fi tables with optional anomaly detection.
- `usecase/wireless_packet_system/README.md`: Wireless Packet System examples, configuration, and supporting assets.

## Notes

- Notebook-based examples continue to live under `notebooks/`, grouped by workflow.
- Command-line examples continue to live under `scripts/`.
- Some use cases may include additional documentation under their own `docs/` directories.
